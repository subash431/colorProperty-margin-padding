LCH color format:

The LCH funtional notion expresses a given color in the LCH color space.
Syntax:
lch(L C H / A )
l: specifies the CIE lightness, and is a percentage between 0% representing black and 100% representing white.
c: species the amount of color and its minimum useful value is 0 and maximum value is theoretically unbounded(but in practice does not exceed 230)
H: is the hue angle
A: species (alpha) can be a number between 0 to 1, or a oercentage from 0% to 100%
